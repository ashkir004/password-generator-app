
# Password Generator App

Secure, configurable password generator built with Sveltekit — adjustable length, character options, strength meter, and copy-to-clipboard. Fully client-side; no backend required.

## Features

- Generate random passwords with configurable length
- Toggle character sets: uppercase, lowercase, numbers, symbols
- Real-time password strength meter and visual feedback
- Copy-to-clipboard with visual confirmation
- Responsive, accessible UI built with Svelte components

## Tech

- Svelte
- Sveltkit
- Vite
- JavaScript


## Quick start

Open a terminal in the project folder and run:

```powershell
npm install
npm run dev
```

Then open the local dev server shown by the CLI (usually http://localhost:5173).

To build for production:

```powershell
npm run build
npm run preview
```

## Project

Key files and folders:

- `src/lib` — password generation logic and helper modules
- `src/lib/components` — Svelte UI components (PasswordDisplay, PasswordOptions, PasswordLength, etc.)
